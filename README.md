### 🧐 42seoul.community ?

**42seoul에서 지원하는 커뮤니티를 위한 기반 사이트!**

- 커뮤니티에 대한 참여 신청 및 개설 신청 기능
- 커뮤니티 활동 기록 아카이빙 기능
- 커뮤니티 관리자를 위한 API제공
